\begin{abstract}

	\acf{AWESOME} is a tool used to evaluate the performance and quality of teaching of modules at Aberystwyth University.
	The aims of this project are to ensure the prototype version of \acs{AWESOME} is security audited, refactored to use better programming practices, fully functioning, and is deployed to capture module evaluation data.
